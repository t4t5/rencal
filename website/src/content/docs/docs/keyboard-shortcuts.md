---
title: Keyboard shortcuts
description: Navigate and control renCal from the keyboard.
---

Press <kbd>?</kbd> in renCal to open the shortcut overlay. On macOS, <kbd>Cmd</kbd> is used where the tables say <kbd>Cmd/Ctrl</kbd>.

## Navigation

| Action                          | Shortcut                          |
| ------------------------------- | --------------------------------- |
| Next day                        | <kbd>→</kbd> or <kbd>L</kbd>      |
| Previous day                    | <kbd>←</kbd> or <kbd>H</kbd>      |
| Next week                       | <kbd>↓</kbd> or <kbd>J</kbd>      |
| Previous week                   | <kbd>↑</kbd> or <kbd>K</kbd>      |
| Next month                      | <kbd>Ctrl</kbd> + <kbd>D</kbd>    |
| Start of this or previous month | <kbd>Ctrl</kbd> + <kbd>U</kbd>    |
| Next event in the agenda        | <kbd>Tab</kbd>                    |
| Previous event in the agenda    | <kbd>Shift</kbd> + <kbd>Tab</kbd> |
| Go to today                     | <kbd>T</kbd>                      |
| Go to a date                    | <kbd>.</kbd>                      |

The go-to-date command accepts phrases such as `5 Sep` or `next Sunday`.

## Views

| Action                | Shortcut     |
| --------------------- | ------------ |
| Month view            | <kbd>M</kbd> |
| Week view             | <kbd>W</kbd> |
| Board view            | <kbd>B</kbd> |
| Switch calendar group | <kbd>G</kbd> |

The group shortcut is active when at least two calendar groups exist.

## General

| Action                           | Shortcut                                                                                |
| -------------------------------- | --------------------------------------------------------------------------------------- |
| Compose an event                 | <kbd>C</kbd>                                                                            |
| Add an event to the selected day | <kbd>A</kbd>                                                                            |
| Search                           | <kbd>/</kbd>, <kbd>Cmd/Ctrl</kbd> + <kbd>F</kbd>, or <kbd>Cmd/Ctrl</kbd> + <kbd>P</kbd> |
| Open command palette             | <kbd>Cmd/Ctrl</kbd> + <kbd>K</kbd>                                                      |
| Open Settings                    | <kbd>Cmd/Ctrl</kbd> + <kbd>,</kbd>                                                      |
| Cycle theme                      | <kbd>Cmd/Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>T</kbd>                                   |
| Show keyboard shortcuts          | <kbd>?</kbd>                                                                            |

## Event shortcuts

- Press <kbd>Enter</kbd> in the quick composer to create the event.
- Press <kbd>Escape</kbd> once to clear the composer, then again to close it.
- Press <kbd>Delete</kbd> or <kbd>Backspace</kbd> while an event is selected to open its deletion confirmation. This is disabled while editing a text field.
