---
title: Creating and editing events
description: Create, update, move, repeat, and delete events in renCal.
---

renCal provides a quick natural-language composer and a detailed event editor.

## Create an event quickly

Select the **+** button or press <kbd>C</kbd>, type an event, and press <kbd>Enter</kbd>.

renCal understands common date and time phrases:

```text
Dentist Friday at 10
Dinner tomorrow at 7 in Soho
Team sync every Monday at 9
Planning every weekday at 10:30
```

Dates, times, recurrence, and a trailing location are highlighted as renCal recognises them. If you do not specify an end time, a timed event lasts one hour.

Press <kbd>Escape</kbd> once to clear the text, then again to close the composer.

## Use the event editor

To create an event with all options visible:

- Press <kbd>A</kbd> to add an event to the selected day.
- Right-click a day in Month view.
- Right-click a time or the all-day area in Week view.

The editor supports:

- timed, all-day, and multi-day events;
- locations and notes;
- calendars and reminders;
- participants;
- daily, weekly, fortnightly, monthly, and yearly recurrence.

New events use the default calendar and reminders configured in **Settings**. You can change either before adding the event.

## Edit an event

Select an event to open it. Changes are saved when the event panel closes. You can edit its title, dates, times, calendar, recurrence, reminders, participants, location, and notes.

Read-only calendars and public subscriptions can be viewed but not edited.

## Recurring events

When you change an occurrence in a recurring series, renCal asks where to apply the edit:

- **Only this event** changes the selected occurrence.
- **This and all future events** starts a new series from the selected occurrence.
- **All events** changes the complete series.

Moving only one occurrence to another calendar is not supported; move the complete series instead.

## Delete events

Open the event menu and select **Delete event**. You can also press <kbd>Delete</kbd> or <kbd>Backspace</kbd> while an event is selected and no text field is active.

For a recurring event, renCal asks whether to delete only the selected occurrence or the complete series.

## Participants and invitations

Enter an email address under **Add participant**. renCal suggests contacts found in your existing calendar events and shows each participant's response status.

New invitations appear as a numbered badge in the toolbar. You can accept or decline there, or open the event to select **Accepted**, **Declined**, or **Maybe**. Conference links supplied by supported providers appear in the event details.
