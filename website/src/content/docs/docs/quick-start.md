This guide takes you from a fresh installation to your first synced event.

## Connect a calendar

When you first open renCal, select **Connect a calendar**.

- Choose **Google**, **iCloud**, **Outlook**, or **Other CalDAV server** to connect an existing account.
- Choose **Local-only calendar** if you want a calendar that never connects to the internet.

The provider will either open a sign-in page or ask for the credentials it needs. See [Accounts and providers](/docs/accounts-and-providers/) for provider-specific details and third-party providers.

## Create an event

Select the **+** button or press <kbd>C</kbd>, then type an event naturally:

```text
Lunch with Sarah tomorrow at 1pm
```

Press <kbd>Enter</kbd> to create it. renCal recognises common dates and times, recurring phrases such as `every Monday`, and locations at the end of a title.

For more control, right-click a day in Month or Week view and select **Create event**. You can then set the calendar, dates, reminders, participants, recurrence, location, and notes before adding the event.

## Navigate your calendar

Use the view selector in the top-right corner to switch between:

- **Month** for an overview of several weeks.
- **Week** for a time-based schedule.
- **Board** for events from yesterday through the end of the current week.

The mini calendar changes the selected date, while **Today** returns to the current day. Press <kbd>?</kbd> at any time to see the available [keyboard shortcuts](/docs/keyboard-shortcuts/).

## Check sync

renCal stores changes locally first and syncs connected calendars automatically by default.

The cloud icon in the toolbar shows the current state. Hover over it to see whether renCal is checking, syncing, waiting for changes, offline, or reporting an error. Select the icon to sync immediately.

Automatic sync can be disabled under **Settings → General** if you prefer to review pending changes and sync manually.
